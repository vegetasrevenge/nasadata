    • NASA Data Mapping Project

    This project uses open datasets from Nasa (for now, meteorites) to allow you access to a visual representation of records obtained.
    The meteorite data itself is quite extensive, with over 45,000 entries ranging from AD860 to 2015 and across the globe.
    Upon logging in, you can actually record data yourself. So give it a go, search for more meteorites and stay tuned as more datasets are obtained!

    Major credits to NASA for such thorough and copious data, all available to the public.
